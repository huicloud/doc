##消息服务
###消息发送
** URL **
qoute/dyna
** 描述 **
动态行情服务
** 参数说明 **

|名称|类型|说明|缺省|
| -------- | -------- | -------- | -------- |
|obj*|字符串|查询股票*表示必填，必填的放在前面如SH600000|无|
|field|字符串|字段过滤(见返回说明)|无|

**结果说明**
```json
{
    "Id": 20,
    "RepDataQuoteDynaSingle": [
        {
            "Obj": "SH601519",
            "Data": {
                "Id": 638231067,//说明

}
```
