
## 行情服务

### 动态行情

**URL**

qoute/dyna

**描述**

动态行情服务

**参数说明**

|名称|类型|说明|缺省|
| -------- | -------- | -------- | -------- |
|obj\*|字符串|查询股票\*表示必填，必填的字段说明放在前面。如SH600000|无|
|field|字符串|字段过滤(见返回说明)|无|
|output|字符串|数据结果的返回格式|json|
