
## 新三板服务

### 财务快报

**URL**

/sanban/quickreport

**描述**

获取财务快报列报协议和做市

**参数说明**

|名称|类型|说明|缺省|
| -------- | -------- | -------- | -------- |
|class\*|整型|0：协议+做市，1：协议，2：做市 \*表示必填|无|
|sort|字符串|输出顺序，当sort=desc时以倒序输出，其他情况下输出为顺序。不指定该字段默认输出为顺序,\*表示必填|无|

```json
    "Counter": 1,
    "Data": {

```

**示例**

[/sanban/quickreport?class=0]($APIHOST$/sanban/quickreport?class=0)
获取财务快报列报协议和做市
