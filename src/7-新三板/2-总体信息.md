
### 总体信息查询

**URL**

/sanban/overallinfo

**描述**

获取总体信息

**参数说明**

无


**结果说明**

```json
{
    "Qid": "",
    "Err": 0,
    "Counter": 1,
    "Data": {
        "Id": 121,
        "RepDataOverallInfo": [
            {
                "TotalStockNum": 5609,
                "AgreementStockNum": 4388,
                "BrokerStockNum": 1221,
                "WaitStockNum": 163
            }
        ]
    }
}
```

**示例**

[/sanban/overallinfo]($APIHOST$/sanban/overallinfo)
获取总体信息
