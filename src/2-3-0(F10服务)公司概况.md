## f10服务

### 公司概况

**URL**

/f10/gsgk

**描述**

公司介绍用

**参数说明**

|名称|类型|说明|缺省|
| -------- | -------- | -------- | -------- |
|obj\*|字符串|查询股票\*表示必填，必填的字段说明放在前面。如SH600000|无|
|field|字符串|字段以都好间隔，为空则返回结构所有字段,关注字段以逗号分隔(见返回说明)|无|
