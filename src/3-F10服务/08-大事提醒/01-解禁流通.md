
### 解禁流通

**URL**

/f10/dstx/jjlt

**描述**

解禁流通

**参数说明**

|名称|类型|说明|缺省|
| -------- | -------- | -------- | -------- |
|obj\*|字符串|查询股票\*表示必填，必填的字段说明放在前面。如SH600000|无|


**结果说明**

```json
{
    "Qid": "",
    "Err": 0,
    "Counter": 1,
    "Data": {
        "Id": 88,
        "RepDataF10DstxJjltOutput": [
            {
                "Obj": "SH600008",
                "Data": [
                    {
                        "date": "2016-01-23 00:00:00",           //日期
                        "jjgf": 21030.7,                         //解禁股份(万股)
                        "zzgf": 8.73,                            //占总股份(%)
                        "gflx": "非公开增发"                      //股份类型 
                    }
                ]
            }
        ]
    }
}
```

**示例**

[/f10/dstx/jjlt?obj=SH600008]($APIHOST$/f10/dstx/jjlt?obj=SH600008)
查询SH600008解禁流通信息
